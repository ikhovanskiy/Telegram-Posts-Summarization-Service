# Frontend - Telegram Posts Summarization

Веб-интерфейс для сервиса суммаризации постов из Telegram-групп.

## Технологии

- **SvelteKit** - фреймворк для создания веб-приложений
- **TypeScript** - типизированный JavaScript
- **Vite** - сборщик и dev-сервер
- **Чистый CSS** - стилизация без фреймворков

## Установка

```bash
npm install
```

## Запуск

### Режим разработки

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:5173

### Сборка для продакшена

```bash
npm run build
```

### Предпросмотр продакшен-сборки

```bash
npm run preview
```

## Структура проекта

```
frontend/
├── src/
│   ├── routes/              # Страницы приложения
│   │   ├── +page.svelte     # Главная страница
│   │   ├── auth/            # Авторизация
│   │   ├── chats/           # Список чатов
│   │   └── summary/         # Суммаризация
│   ├── lib/                 # Библиотеки и утилиты
│   │   ├── api.ts           # API клиент
│   │   └── types.ts         # TypeScript типы
│   └── app.html             # HTML шаблон
├── static/
│   └── styles/
│       └── app.css          # Глобальные стили
└── package.json
```

## Конфигурация

Убедитесь, что backend API запущен на `http://localhost:3000`. Если используется другой адрес, измените `API_URL` в файле `src/lib/api.ts`.

## Основные страницы

- `/` - Главная страница с кнопкой входа
- `/auth` - Авторизация через Telegram
- `/chats` - Список доступных чатов
- `/summary/[chatId]` - Создание суммаризации для выбранного чата

## Разработка

Все ошибки TypeScript исчезнут после установки зависимостей командой `npm install`.
